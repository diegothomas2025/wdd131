Site Name

THOMAS WEB

I chose this name because it represents my personal brand as a developer. The name is simple, professional, and easy to remember. It reflects the purpose of the site: providing modern and responsive web design services for startups and small businesses.

Site Purpose

The purpose of THOMAS WEB is to showcase my web development skills and offer simple and affordable services to small businesses and entrepreneurs. The site will desmostrate examples of resposive websites built with HTML, CSS, and JavaScript, and will also provide a contact section for potencial clients who want to develop their online presence.

Scenarios

How much does it cost to create a simple website for my small business?

Can I see examples of responsive websites you have built?

Color Schema

Primary color: #1E3A8A (Blue) - headings, important links, main buttons.

Secondary color: #F3F4F6 (Light Gray) - backgrounds.

Text color: #374151 (Dark Gray) - texts

Accent color: #F97316 (Orange) - important icons or buttons

Typography

Body: Roboto
